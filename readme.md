## Instalação

Via Composer

``` bash
$ composer require wandsdev/laravel-auth-api
```

## Uso

``` bash
$ php artisan migrate
```

``` bash
php artisan vendor:publish --provider="WandsDev\AuthApi\AuthApiServiceProvider"
```
